### vmdevirt
```sh
git clone https://git.back.engineering/vmp3/vmdevirt.git --recurse-submodules --shallow-submodules
cd vmdevirt
export CC=/usr/bin/clang
export CXX=/usr/bin/clang++
mkdir include
cmake -B build
cd build
make
```
需要编译 LLVM。
